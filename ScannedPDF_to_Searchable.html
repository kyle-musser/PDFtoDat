This notebook will go through an example of converting a folder of scanned pdf's into searchable pdfs.

NOTE: This tutorial assumes your are using Windows.

STEP 1 : Install Tesseract OCR¶

  • The first step will be to install the Tesseract OCR engine on your machine: https://github.com/UB-Mannheim/tesseract/wiki

  • The Tesseract engine will recognize english characters and convert the scanned pdf's into text. Note that the engine is not 100% accurate and accuracy may vary.

  • Select the 32 or 64 bit version based on your windows machine. (To find if your computer is 32 or 64 bit search "System Information" in windows search and look for System Type.

  • NOTE : It is best to install into the predefined folder during instillation. This folder is usually "Programs/Tesseract-OCR"

STEP 2 : Install Python¶

  • The easiest way to install python is through Anaconda:

    • https://www.anaconda.com/products/individual

  • Again choose the 64 or 32 bit windows version depending on your machine

Step 3 : Install Packages in Python¶

For reference see: https://datatofish.com/how-to-install-python-package-in-anaconda/

  • In Windows search bar type : Anaconda Prompt

  • We then need to pip install several packages as show below, make sure each individual package downloads before installing the next one.

  • pip install Pillow
  • pip install pytesseract
  • pip install pdf2image
  • pip install PyPDF2

Step 4.1 : Convert our PDFs¶

To do this we will open "Spyder within Anaconda"

  1. Open Anaconda
  2. Click "Launch" next to Spyder. You should see this IDE load if done correctly:

Step 4.2 : Insert the Code below¶

  • And adjust so the folder paths match your machine

  • You should not need to edit any lines of code after setting the indir, outdir and tesseract_dir

  • Then you just need to hit "Run" in Spyder and you should see a list of files being converted in the console.

#Importing packages similar to R
#NOTE: make sure they are installed! 
from PIL import Image
import pytesseract
from pdf2image import convert_from_path
import os
import PyPDF2

# Get list of PDF in folder of choice
from os import listdir
from os.path import isfile, join

#Set full folder path to folder with scanned PDFs
#include "r" at the begenning and only use forward (/) slashes in between
indir = r"C:/Users/kylem/Desktop/convertPDF/scanned_pdf"

#SEt folder path for output (This should be a seperat folder than the one above)
outdir = r"C:/Users/kylem/Desktop/convertPDF/searchable_pdf"

#Set location of tesseract.exe we just downloaded
tesseract_dir =  r"C:\Program Files\Tesseract-OCR\tesseract.exe"

#This command get a list of all files in the "indir" folder
# SO make sure this folder only includes the scanned PDFs!
pdfs = [f for f in listdir(indir) if isfile(join(indir, f))]

#SHOULD NOT HAVE TO EDIT ANY CODE BELOW THIS LINE
for pdf in pdfs:
    print(pdf)
    # 1 : Converting each PDF page to images
    # Store all the pages of the PDF in a variable
    os.chdir(indir)
    pages = convert_from_path(pdf)

    # Counter to store images of each page of PDF to image
    image_counter = 1

    # Iterate through all the pages stored above
    for page in pages:
        os.chdir(outdir)
        # remove .pdf extension from filename
        pdfname = pdf.replace(".pdf", "")

        # Declaring filename for each page of PDF as JPG
        filename = str(pdfname)+str(image_counter)+".jpg"

        # Save the image of the page in system
        page.save(filename, 'JPEG')

        # Increment the counter to update filename
        image_counter = image_counter + 1

    # 2 - Recognizing text from the images using OCR
    # Variable to get count of total number of pages
    filelimit = image_counter-1

    # Iterate from 1 to total number of pages
    for i in range(1, filelimit + 1):
        print(i)

        pdfname = pdf.replace(".pdf", "")

        #make file names
        filename_jpg = str(pdfname)+str(i)+".jpg"
        filename_pdf = str(pdfname) + "_" + "pg" + str(i) + ".pdf"
        print(filename_pdf)

        # Convert PDF to searchable PDF to use with Tabula
        pytesseract.pytesseract.tesseract_cmd = tesseract_dir
        pdf_wSText = pytesseract.image_to_pdf_or_hocr(Image.open(filename_jpg), extension='pdf')
        f = open(filename_pdf, 'w+b')
        f.write(bytearray(pdf_wSText))
        f.close()


    #Make PDF Pages to combine all pages into the original PDF, but searchable.
    mergedpdf = PyPDF2.PdfFileMerger()
    for i in range(1, filelimit + 1):
        filename_pdf = str(pdfname) + "_" + "pg" + str(i) + ".pdf"
        mergedpdf.append(PyPDF2.PdfFileReader(filename_pdf, 'rb'))

    mergedpdf.write(pdf)

Step 5 : Cleanup "Outdir"¶

Once the code runs, in the "outdir" you set, you will have three types of files:

  1. JPEGS of each pdf page (You can delete all JPEGS)
  2. Individual pages of all PDFs (these have pg. number extensions at the end of the file)
  3. Recombined PDFs that are now searchable. (these dont have page number extensions at the end of the file) You should move the recombined PDFs (i.e. the PDF without page numbers at the end) to another folder that is not yor outdir.

So to summarize, after the code runs and you cleanup the folder the outdir should be left with only individual PDF pages that you want converted into data. Pages that dont have any information can be deleted or moved to another folder for backup.
